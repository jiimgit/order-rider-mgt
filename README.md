# Order Rider Management Platform

A delivery platform with MLM commission structure built with Next.js and Supabase.

## Features

- Customer Portal: Post delivery jobs, manage credits
- Rider Portal: Accept jobs, track earnings, referral system
- Admin Portal: Manage customers, riders, and jobs

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
